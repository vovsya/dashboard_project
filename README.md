# Приложение-дашборд
    Пользователь может создавать страницы и добавлять на них виджеты погоды, курса валют, список дел, время, пробки и тд
Деплой https://dashboard-project-vovsya.onrender.com

Асинхронный стэк:

    Python 3 (FastAPI + SQLAlchemy async/sync + миграции Alembic)
    
    PostgreSQL + asyncpg
    
    asyncio, HTTPX для асинхронных эндпоитов и запросов в БД
    
    Авторизация OAuth2, JWT-токены
